HTML

Vamos a hablar sobre HTML, el lenguaje de marcado de hipertexto que es utilizado para crear y estructurar el contenido de las páginas web.
HTML es un lenguaje de marcado, lo que significa que utiliza etiquetas para definir la estructura y el contenido de una página web. Las etiquetas están rodeadas por corchetes angulares <>, y algunas etiquetas tienen atributos que se utilizan para proporcionar información adicional sobre el contenido.
Es importante recordar que HTML sólo se utiliza para definir la estructura y el contenido de una página web. Para darle estilo a la página y hacerla más atractiva visualmente, se utiliza CSS (hojas de estilo en cascada) y para agregar interactividad y dinamismo, se utiliza JavaScript.
En resumen, HTML es la base de cualquier página web y es importante comprender su sintaxis y cómo se utiliza para crear contenido en la web. A medida que avancemos, exploraremos cómo utilizar CSS y JavaScript para hacer que las páginas web sean aún más atractivas y dinámicas. ¡Manos a la obra!

Etiqueta HTML

Las etiquetas HTML se utilizan para definir y estructurar el contenido de una página web. Las etiquetas se colocan entre los símbolos "<" y ">" , y a menudo incluyen atributos que especifican cómo se debe presentar el contenido.

Atributos HTML

Los atributos HTML se utilizan para especificar cómo se debe presentar el contenido de una etiqueta HTML. Los atributos se incluyen en la etiqueta y tienen un formato de nombre/valor. Algunos ejemplos comunes de atributos incluyen "class", "id", "href", "src"...


Encabezados

Los encabezados se utilizan para definir la estructura de la página y proporcionar una jerarquía visual para el contenido. Hay seis niveles de encabezados, que se definen con las etiquetas "h1" a "h6".


Párrafos

Los párrafos se utilizan para definir bloques de texto. Se definen con la etiqueta "p".

Enlaces

Los enlaces se utilizan para vincular una página a otra o a una ubicación específica en una página. Los enlaces se definen con la etiqueta "a";, y se especifica la dirección del enlace con el atributo "href".

Imágenes

Las imágenes se utilizan para agregar gráficos yotro contenido visual a una página web. Las imágenes se definen con la etiqueta "img", y se especifica la ubicación de la imagen con el atributo "src".

Listas

Las listas se utilizan para presentar información en formato de lista, ya sea como una lista ordenada ("ol") o una lista no ordenada ("ul"). Cada elemento de la lista se define con la etiqueta "li".

Tablas

Las tablas se utilizan para presentar información en formato de tabla, con filas y columnas. Las tablas se definen con la etiqueta "table", y se especifica cada fila con la etiqueta "tr" y cada celda con la etiqueta "td" o "th".

Formularios

Los formularios se utilizan para recopilar información del usuario a través de campos de entrada. Los formularios se definen con la etiqueta "form", y se especifica cada campo de entrada con una variedad de etiquetas, como "input", "textarea", "select", etc.

IFrames

Los IFrames se utilizan para insertar contenido de otra página web en una página. Los IFrames se definen con la etiqueta "iframe", y se especifica la ubicación del contenido con el atributo "src".
